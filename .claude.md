### Agent Framework Patterns
The project uses Azure AI Foundry hosted agents with the following SDK patterns:

```python
from azure.ai.projects.aio import AIProjectClient
from azure.identity import DefaultAzureCredential

# Initialize client
project = AIProjectClient(
    endpoint="https://<resource>.services.ai.azure.com/api/projects/<project>",
    credential=DefaultAzureCredential(),
)

# For hosted agents - use get_openai_client() for inference
openai_client = project.get_openai_client()

# For creating runtime agents - use agents API
agent = project.agents.create_agent(model="gpt-4o", name="my-agent", instructions="...")
thread = project.agents.threads.create()
project.agents.messages.create(thread_id=thread.id, role="user", content="...")
run = project.agents.runs.create_and_process(thread_id=thread.id, agent_id=agent.id)
```
